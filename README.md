# json_to_dart

A web and desktop app to convert json to dart class model.

visit here to try : [link](https://stha-ums.github.io/json-dart/)

### Features
- [x]  toJson
- [x] fromJSon
- [x] copyWith method
- [x] option for mutability
- [x] use Equatable
- [x] use Freezed
- [x] use json_annotation
